# 🚀 启动开发服务器

## ❌ 错误：ERR_CONNECTION_REFUSED

这个错误表示开发服务器没有运行。

---

## ✅ 解决步骤

### 1. 打开终端

打开终端（Terminal）应用。

### 2. 进入项目目录

```bash
cd /Users/jojodd/my-tool-platform
```

### 3. 检查依赖

如果 `node_modules` 目录不存在，需要先安装依赖：

```bash
npm install
```

### 4. 启动开发服务器

```bash
npm run dev
```

### 5. 等待启动完成

你会看到类似这样的输出：

```
▲ Next.js 14.2.5
- Local:        http://localhost:3001
- Ready in 2.3s
```

### 6. 访问平台

看到 "Ready" 后，打开浏览器访问：

**http://localhost:3001**

---

## ⚠️ 如果启动失败

### 问题1：端口被占用

```bash
# 查看占用端口的进程
lsof -i :3001

# 或使用其他端口
npm run dev -- -p 3002
```

### 问题2：依赖未安装

```bash
npm install
```

### 问题3：权限问题

```bash
sudo npm install
```
