## 🔍 在豆包控制台查找模型端点

### 步骤 1：登录控制台

访问：https://console.volcengine.com/

### 步骤 2：查找模型端点

在控制台中查找以下信息：

1. **模型服务页面**：
   - 找到"模型服务"或"API 服务"
   - 查看你的模型端点列表
   - 找到端点ID（通常是 `ep-` 开头）或模型名称

2. **API 调用示例**：
   - 查看控制台中的 API 调用示例
   - 查看文档中的模型名称格式

3. **服务列表**：
   - 查看"我的服务"或"服务列表"
   - 找到可用的模型端点

### 步骤 3：配置模型名称

找到模型端点后，在 `.env.local` 文件中添加：

```bash
# 如果是端点ID格式（ep-开头）
DOUBAO_MODEL=ep-你的端点ID

# 如果是模型名称格式
DOUBAO_MODEL=你的模型名称
```

## 📝 常见问题

### Q: 控制台中没有看到模型端点？
A: 可能需要：
- 创建模型服务
- 部署模型端点
- 查看"服务管理"或"端点管理"页面

### Q: API Key 格式是什么？
A: 你的 API Key 是 UUID 格式，这是正确的。需要配合模型端点或模型名称使用。

### Q: 如何知道使用哪个模型？
A: 查看控制台中的：
- 模型列表
- 服务列表
- API 调用示例
- 文档中的模型名称
