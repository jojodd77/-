## 🏗️ 技术架构

### 前端
- **框架**：Next.js 14 (App Router)
- **语言**：TypeScript
- **样式**：Tailwind CSS
- **组件**：React Hooks

### 后端
- **API**：Next.js API Routes
- **路由**：`/api/correct` - 发音修正接口

### 数据存储
- **历史记录**：localStorage（当前）
- **可升级**：PostgreSQL / Supabase / MongoDB
