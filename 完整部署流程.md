## 🎯 第二步：部署到 Vercel

### 1. 访问 Vercel

打开浏览器：**https://vercel.com**

### 2. 登录/注册

- 点击 "Sign Up" 或 "Log In"
- 选择 **"Continue with GitHub"**
- 授权 GitHub 访问

### 3. 导入项目

1. 点击 **"+ New Project"**
2. 在 "Import Git Repository" 中找到你的仓库
   - 仓库：`jojodd77/-`
   - 如果看不到，点击 "Adjust GitHub App Permissions" 授权
3. 点击 **"Import"**

### 4. 配置项目

Vercel 会自动检测 Next.js，通常不需要修改：

- **Framework Preset**: Next.js ✅
- **Root Directory**: `./` ✅
- **Build Command**: `npm run build` ✅

### 5. 配置环境变量（重要！）

在 "Environment Variables" 部分：

1. 点击 **"Add"**
2. 添加：
   - **Key**: `ZHIPU_API_KEY`
   - **Value**: `dff3af0865ac4a75b7511dd22d6188b8.JBxKqerah1FuJVba`
   - **Environment**: 全部勾选（Production, Preview, Development）
3. 点击 **"Add"** 保存

### 6. 开始部署

1. 点击 **"Deploy"** 按钮
2. 等待构建完成（2-5 分钟）
3. 看到 **"Congratulations!"** 表示成功！
