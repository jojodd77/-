# 🧪 测试大模型修正功能

## 📝 测试用例

请用以下文本测试，验证大模型是否正确学习：

### 测试1：基本多音字
**输入**：图中哪个光路图是正确的？
**期望输出**：图中(/zhong1/)哪个光路图是正确的？

### 测试2：不同语境
**输入**：直接选中了答案
**期望输出**：直接选中(/zhong4/)了答案

### 测试3：解字
**输入**：让大家流口水解渴的？
**期望输出**：让大家流口水解(/jie3/)渴的？

### 测试4：差字（多次出现）
**输入**：其实这个速度差异有个专门的名称，叫做速度差。
**期望输出**：其实这个速度差(/cha1/)异有个专门的名称，叫做速度差(/cha1/)。

---

## 🔍 如果结果不对

1. **检查 API Key**：确认 `.env.local` 中的 API Key 正确
2. **查看日志**：检查终端中的错误信息
3. **测试 API**：直接测试智谱清言 API 是否正常
4. **优化 Prompt**：可能需要进一步简化 Prompt

---

## 💡 调试方法

在 `app/api/correct/route.ts` 中添加日志：

```typescript
console.log('发送给大模型的 Prompt:', prompt);
console.log('大模型返回:', content);
```

这样可以查看大模型实际接收和返回的内容。
