# ⚡ 快速修复 500 错误

## 🎯 一键修复（推荐）

运行以下命令：

```bash
cd /Users/jojodd/my-tool-platform
./修复500错误.sh
```

## 📋 手动修复步骤

### 1. 停止当前服务器

```bash
# 查找并停止占用 3001 端口的进程
lsof -ti :3001 | xargs kill -9
```

### 2. 清理构建缓存

```bash
cd /Users/jojodd/my-tool-platform
rm -rf .next
```

### 3. 确保环境变量文件存在

```bash
# 如果 .env.local 不存在，创建它
if [ ! -f .env.local ]; then
    cp env.example .env.local
fi
```

### 4. 重新启动开发服务器

```bash
npm run dev
```

## ✅ 验证修复

1. 访问 http://localhost:3001
2. 页面应该正常显示，不再是空白
3. 浏览器控制台（F12）没有红色错误
