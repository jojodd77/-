## 📋 交接步骤

### 对于接收方

1. **获取代码**
   ```bash
   # 方式1: 从 Git 仓库克隆
   git clone <repository-url>
   git checkout v1.0.0
   
   # 方式2: 解压压缩包
   tar -xzf my-tool-platform-v1.0.0.tar.gz
   ```

2. **安装依赖**
   ```bash
   cd my-tool-platform
   npm install
   ```

3. **配置环境变量**
   ```bash
   cp env.example .env.local
   # 编辑 .env.local 填入 API 密钥
   ```

4. **运行项目**
   ```bash
   npm run dev
   # 访问 http://localhost:3001
   ```

5. **阅读文档**
   - 先阅读《项目交接文档.md》
   - 查看 README.md 了解基本功能
   - 参考 env.example 配置 API
