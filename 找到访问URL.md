# 🔍 找到访问 URL

## 🎯 方法1：在 Dashboard 查看

1. **点击 "Continue to Dashboard" 按钮**
2. **在 Dashboard 中**：
   - 找到你的项目卡片
   - 项目名称下方会显示 URL
   - 或点击项目进入详情页

3. **在项目详情页**：
   - 找到 **"Domains"** 部分
   - 你会看到：`https://pronunciation-correction-platform.vercel.app`
   - 这就是你的访问地址！

---

## 🎯 方法2：在部署页面查看

如果还在部署完成页面：
- 查看页面顶部
- 应该能看到 **"Visit"** 或 **"View"** 按钮
- 点击即可访问

---

## 🔑 重要：配置环境变量

如果还没配置环境变量：

1. **进入项目 Settings**
2. **找到 "Environment Variables"**
3. **添加**：
   ```
   Key: ZHIPU_API_KEY
   Value: dff3af0865ac4a75b7511dd22d6188b8.JBxKqerah1FuJVba
   Environment: 全部勾选
   ```
4. **保存后重新部署**

---

## 🌐 访问 URL 格式

你的 URL 通常是：
```
https://pronunciation-correction-platform.vercel.app
```

或：
```
https://pronunciation-correction-platform-[你的用户名].vercel.app
```
