# 🚀 开发地址配置

## 📍 本地开发地址

**新平台开发地址**：`http://localhost:3001`

### 为什么使用 3001？

- ✅ 避免与现有项目（annotation-platform）的 3000 端口冲突
- ✅ 可以同时运行多个项目
- ✅ 更清晰的端口区分

```bash
cd /Users/jojodd/my-tool-platform

npm run dev
```

然后访问：**http://localhost:3001**

---

## 🔧 端口配置

### 当前配置

- **开发服务器端口**：`3001`
- **配置文件**：`package.json` 中的 `dev` 脚本
- **环境变量**：`env.example` 中的 `NEXT_PUBLIC_APP_URL`

### 如需修改端口

编辑 `package.json`：

```json
{
  "scripts": {
    "dev": "next dev -p 你的端口号"
  }
}
```
