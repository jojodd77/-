## 🐛 如果遇到问题

### 问题 1：重启后仍然显示"未配置"

**解决方法**：
1. 确认 `.env.local` 文件在项目根目录
2. 检查配置格式是否正确（无空格、无引号）
3. 确认服务器已完全重启

### 问题 2：API 调用失败

**可能原因**：
- API Key 无效
- 额度已用完
- 网络问题

**解决方法**：
1. 检查终端中的详细错误信息
2. 登录智谱清言控制台检查 API Key 状态
3. 查看是否有剩余额度
