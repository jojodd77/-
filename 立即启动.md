# 🚀 立即启动开发服务器

## 📋 快速启动步骤

### 1. 打开终端

打开终端（Terminal），执行以下命令：

### 2. 进入项目目录

```bash
cd /Users/jojodd/my-tool-platform
```

### 3. 安装依赖（如果还没安装）

```bash
npm install
```

**如果遇到权限问题**，可以尝试：
```bash
sudo npm install
```

或者检查 npm 配置：
```bash
npm config get prefix
```

### 4. 启动开发服务器

```bash
npm run dev
```

### 5. 访问平台

看到类似这样的输出后：

```
▲ Next.js 14.2.5
- Local:        http://localhost:3001
- Ready in 2.3s
```

打开浏览器访问：**http://localhost:3001**

## ✅ 验证功能

1. **首页**：显示"发音修正平台"和三个功能卡片

## ⚠️ 如果遇到问题

### 问题1：端口 3001 已被占用

```bash
# 查看占用端口的进程
lsof -i :3001

# 或使用其他端口
npm run dev -- -p 3002
```

### 问题2：npm install 失败

```bash
# 清除缓存
npm cache clean --force

# 删除 node_modules 重新安装
rm -rf node_modules package-lock.json
npm install
```

### 问题3：依赖安装很慢

可以使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```
