## 🔍 如何判断安装是否完成？

### 方法一：查看最后一行

如果看到类似这样的输出：

```
added 250 packages, and audited 251 packages in 2m
```

或者：

```
found 0 vulnerabilities
```

**说明安装已完成！** ✅

### 方法二：检查 node_modules 目录

```bash
ls -la node_modules | head -5
```

如果能看到很多文件夹，说明安装成功。

### 方法三：查看 package-lock.json

```bash
ls -la package-lock.json
```

如果文件存在，说明安装完成。

## ⚠️ 如果看到错误（不是警告）

如果看到 `npm ERR!` 开头的错误（不是 `npm warn`），那才需要处理。

常见的错误：
- `npm ERR! code EACCES` - 权限错误
- `npm ERR! network` - 网络错误
- `npm ERR! code 1` - 安装失败
